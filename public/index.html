概要

本APIでは、 NTT Communications OCNギャランティサービスに関するリソースの参照・操作が可能です。

  • 同一グローバルIPからの同時アクセス: X リクエスト/秒
  • 同一アカウントからの同時アクセス: Yリクエスト/秒
  • 稼働時間: 24/365

ご利用にあたって

  • APIは機能追加・改修のためにバージョンアップが行われます。
    • 旧バージョンのAPI廃止は、事前周知の上で行われます。
    • API アクセスにはアカウントの作成と <とある認証方式> による認証が必要です。
    • アカウント作成や <とある認証方式> の利用方法、その他 API については、 <とあるAPI認証ガイド> をご参照下さい。
    • レスポンスボディにおいて、空配列、空オブジェクト、null リテラルは再帰的にプロパティごと除去されます。 スキーマで required と定義されていないプロパティが対象となるので、その項目をご確認の上ご利用ください。
